# Ideaship
An AI website generator project I worked on in early 2025. Basically a Loveable/Vercel v0 clone.

https://github.com/user-attachments/assets/daff7530-8fb3-4f1b-bab4-a8a2ae8f4cfa

Live version deployed [here](https://ideaship-git-main-nathannlus-projects.vercel.app/)

## Features
- Generate frontend applications with no code
- No-code editor allowing users to easily change AI generated features
- Saves progress as git commits


## Setup
1. Download the repository
2. Clone env.example -> .env and fill in with your own
3. `npm run dev`
